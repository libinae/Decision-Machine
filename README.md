# 🎯 多重人格决策机

> 让5种人格为你展开理性与感性的精彩辩论，助你做出最适合自己的选择

## 🌟 项目简介

**多人格决策** 是一款基于多智能体AI的辩论式决策辅助工具。当你面临两难抉择时，系统会创建5种具有不同思维模式的人格化身，让它们围绕决策主题展开激烈辩论，最终由综合人格担任裁判给出专业点评。

## 🎭 5大人格

| 人格 | 图标 | 特点 | 决策倾向 |
|------|------|------|----------|
| 冒险人格 | 🚀 | 大胆、激进、相信风险与收益成正比 | 支持激进选项 |
| 保守人格 | 🛡️ | 谨慎、稳健、重视风险控制 | 支持保守选项 |
| 感性人格 | ❤️ | 温暖、共情、重视内心感受 | 被情感因素影响 |
| 理性人格 | 🧠 | 冷静，分析，用数据说话 | 被逻辑说服 |
| 综合人格 | 🔮 | 平衡全局、客观中立 | 担任裁判 |

## 📋 辩论流程

```
┌────────────────────────────────────────────────────────────┐
│  阶段一：讨论与分组                                        │
├────────────────────────────────────────────────────────────┤
│  ① 5个人格依次表态，选择支持正方或反方                    │
│  ② 综合人格分析立场，确定分组                              │
└────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────┐
│  阶段二：正式辩论                                         │
├────────────────────────────────────────────────────────────┤
│  ① 开篇陈词（正方一辩 → 反方一辩 → 反方二辩 → 正方二辩） │
│  ② 自由辩论（每方最多10次发言）                          │
│  ③ 双方结辩                                               │
│  ④ 裁判总结与裁决                                          │
└────────────────────────────────────────────────────────────┘
```

## ✨ 核心特性

### 🎨 视觉设计
- 统一美观的终端UI边框
- 正方发言浅绿色背景
- 反方发言浅红色背景
- 清晰的阶段分隔
- 适合录屏展示

### 🧠 智能辩论
- **真正的流式输出** - Token级实时显示，已修复重复输出问题
- **上下文理解** - 辩手能看到完整对话历史
- **人格一致性** - 每个角色保持独特风格
- **数据支撑** - 理性人格会用数据讲故事
- **情感共鸣** - 感性人格从普通人视角发声

### ⚡ 高效执行
- 全程约2-3分钟
- 20轮自由辩论（每方10次）
- 10轮后自动打断进入结辩

## 🚀 快速开始

### 环境要求
```bash
pip install agentscope pydantic
```

### 配置API
在项目根目录创建 `.env` 文件（参考 `.env.example`）：
```bash
cp .env.example .env
# 编辑 .env，填入你的 API Key
```

### 运行程序
```bash
# 设置环境变量
export DASHSCOPE_API_KEY="your-api-key"

# 运行辩论
python -m cli
```

或直接运行：
```bash
python cli/main.py
```

### 配置参数
可通过环境变量配置：
```bash
export DASHSCOPE_API_KEY="your-api-key"
export DM_MODEL_NAME="qwen-plus"        # 可选，默认 qwen-plus
export DECISION_MAX_ROUNDS=10           # 可选，默认 10
```

## 🔧 扩展定制

### 修改辩题
在代码中修改 `DEBATE_TOPIC` 变量：
```python
DEBATE_TOPIC = "我应该去大城市发展还是留在家乡？"
```

### 调整辩论轮次
修改 `max_per_side` 参数：
```python
max_per_side = 15  # 每方15次发言
```
